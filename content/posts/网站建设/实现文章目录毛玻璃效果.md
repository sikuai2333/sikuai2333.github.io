# 毛玻璃效果

```styl
// 文件路径 themes\butterfly\source\css\_layout\aside.styl

.toc-item
	  &.active
		.toc-child
		  display: block
	  &:not(.active)      
		// 蒙版模糊 毛玻璃
		-webkit-filter:blur(2px);
```

# 思路

找到样式文件，在\_layout文件夹，然后添加&:not(.active)的样式即可
